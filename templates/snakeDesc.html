Snake Game in C with Ripes

The Snake Game project is an implementation of the classic Snake game in C, designed to be used with the Ripes software. Ripes is a visual computer architecture simulator that enables users to visualize the execution of assembly code on a RISC-V processor.

In this project, I developed a simple yet engaging version of the Snake game. The game logic is written in C, and the Ripes simulator is used to visualize the execution of the game's code. This project not only demonstrates my proficiency in C programming but also my ability to work with hardware simulators and understand computer architecture concepts.

Project Features

  • Implemented the classic Snake game in C.
  • Utilized the Ripes simulator for visualizing code execution.
  • Designed to help understand the execution of assembly code on a RISC-V processor.
  • Interactive gameplay with a graphical user interface.

Technologies Used

  • C Programming Language
  • Ripes Simulator
  • RISC-V Assembly

Project Challenges

One of the main challenges of this project was integrating the game logic written in C with the Ripes simulator. This required a deep understanding of both the C programming language and the RISC-V architecture. Additionally, ensuring that the game runs smoothly within the constraints of the simulator posed another challenge that I successfully overcame.

Project Outcome

The project was a success, resulting in a fully functional Snake game that can be used to visualize the execution of assembly code on a RISC-V processor. This project helped solidify my understanding of computer architecture and assembly language programming.
